To start the game input Start.
To restart the game input Restart.
To make a move input Move, than input what move you want to make. 
Example: Move
         a2a4
To make a castling input Castling, than choose White or Black, than choose Long or Short.
To save a game input Save and name of file.
Example: Save
         test.txt
To load a game input Load and name of file.
Example: Load
         test.txt
If checkmated you will see "Game over!".
If you want restart game input "Restart".
